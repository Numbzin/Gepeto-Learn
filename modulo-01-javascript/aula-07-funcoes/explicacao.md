# Aula 07 – Funções em JavaScript

## O que é uma função?
Um bloco de código reutilizável que executa uma tarefa.

## Tipos:
- Sem parâmetro
- Com parâmetro
- Com retorno

## Exemplo:
```js
function saudacao() {
  console.log("Olá!");
}

function somar(a, b) {
  return a + b;
}
```

## Por que usar funções?
- Evita repetição
- Organização
- Clareza
