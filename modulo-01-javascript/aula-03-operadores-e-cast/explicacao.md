# Aula 03 – Operadores + Conversão de Tipos

## Operadores matemáticos:
- +, -, *, /, %, **

## Conversão de tipos:
- Number("10")
- String(123)
- parseInt(), parseFloat()
- toString()

## Template String:
```js
const nome = "Ana"
const idade = 20
console.log(\`Meu nome é \${nome} e tenho \${idade} anos\`)
```
