# Aula 06 – Objetos em JavaScript

## O que é um objeto?
É uma estrutura que armazena dados no formato de pares chave: valor.

## Exemplo:
```js
const aluno = {
  nome: "Ana",
  idade: 17,
  nota1: 8,
  nota2: 7
}
```

## Ações comuns:
- Acessar: `aluno.nome`
- Modificar: `aluno.idade = 18`
- Adicionar: `aluno.media = 7.5`
- Remover: `delete aluno.idade`
